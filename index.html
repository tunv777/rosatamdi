Chi Tiet Tinh Nang

Mo ta chi tiet cac tinh nang chinh cua FamilyGuard

🔒 Device Lock/Unlock

  • 📌 Khoa man hinh tu xa ngay lap tuc
  • 📌 Mo khoa tu xa hoac theo lich hen gio
  • 📌 Chon che do khoa: hoan toan / chi cho phep goi khan cap / chi hoc tap
  • 📌 Tu dong khoa theo lich (gio ngu, gio hoc)
  • 📌 Con co the gui yeu cau mo khoa cho cha me

📱 App Management

  • 📌 Xem danh sach tat ca app da cai tren thiet bi con
  • 📌 Cho phep / Chan tung app cu the
  • 📌 Gioi han thoi gian su dung moi app
  • 📌 Phan loai app theo nhom (Hoc tap, Giai tri, MXH...)
  • 📌 Duyet yeu cau cai app moi tu con
  • 📌 Chan go cai dat app FamilyGuard

🌐 Web Filtering

  • 📌 Whitelist - Chi cho phep website cha me duyet
  • 📌 Blacklist - Chan website cu the
  • 📌 Loc theo danh muc (Giao duc, Giai tri, MXH, 18+...)
  • 📌 Safe Search tren Google, Bing, YouTube
  • 📌 Chan quang cao khong phu hop
  • 📌 AI-powered content detection (nhan dien noi dung xau)

📊 Usage Monitoring

  • 📌 Bao cao thoi gian su dung theo ngay / tuan / thang
  • 📌 Top app su dung nhieu nhat
  • 📌 Lich su web da truy cap
  • 📌 So sanh thoi gian hoc tap vs giai tri
  • 📌 Xu huong su dung (tang / giam)
  • 📌 Email bao cao hang ngay cho cha me

📍 Location Tracking

  • 📌 Vi tri real-time tren ban do
  • 📌 Tao vung an toan (Geofence) - Nha, Truong, Lop hoc them
  • 📌 Thong bao khi con vao/roi vung an toan
  • 📌 Lich su di chuyen trong ngay
  • 📌 Nut SOS khan cap cho con

⏰ Schedule & Rules

  • 📌 Tao lich su dung theo ngay trong tuan
  • 📌 Che do Gio hoc (chi app hoc tap)
  • 📌 Che do Gio ngu (khoa hoan toan)
  • 📌 Che do Cuoi tuan (noi long hon)
  • 📌 Che do Ky thi (siet chat gioi han)
  • 📌 Template lich co san theo do tuoi

📱 Mobile Tech Stack

  • Framework React Native (cross-platform)
  • State Zustand / Redux Toolkit
  • Navigation React Navigation 6
  • Push Firebase Cloud Messaging (FCM)
  • Maps React Native Maps / Google Maps
  • Native Module Device Policy Manager (Android) / MDM (iOS)
  • Realtime Socket.io cho lock/unlock tuc thoi

🖥️ Backend Tech Stack

  • Runtime Node.js + TypeScript
  • Framework NestJS (microservices)
  • Database PostgreSQL (primary) + MongoDB (logs)
  • Cache Redis (sessions, realtime state)
  • Queue Bull / RabbitMQ cho background jobs
  • Auth JWT + Refresh Token + PIN
  • Analytics ClickHouse cho bao cao

🔐 Bao Mat

  • 📌 End-to-end encryption cho du lieu vi tri
  • 📌 JWT + Refresh Token rotation
  • 📌 Parent PIN code xac thuc tren child device
  • 📌 HTTPS/TLS cho moi request
  • 📌 Device attestation (chong fake device)
  • 📌 COPPA/GDPR compliance cho du lieu tre em

⚡ Android-Specific

  • Device Admin DevicePolicyManager API
  • Usage UsageStatsManager
  • Lock Accessibility Service + kiosk mode
  • VPN Local VPN cho URL filtering
  • Background Foreground Service (luon chay)
  • Anti-uninstall Device Owner mode

🍎 iOS-Specific

  • MDM Mobile Device Management profile
  • Screen Time Screen Time API (iOS 15+)
  • Filter Content Filter / Network Extension
  • Family FamilyControls framework
  • Managed Managed App Configuration
  • Note iOS han che hon Android, can MDM enrollment

🔄 Communication Flow

  • Command Parent gui lenh → API → Push → Child thuc thi
  • Report Child gui usage data moi 5 phut
  • Lock WebSocket cho lock/unlock (<2s delay)
  • Alert Child trigger → API → Push → Parent nhan
  • Location GPS update moi 1-5 phut (tuy pin)
